# Security Policy

## Supported Versions

Only the latest version of the Zerodha Brokerage Calculator is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.2.x  | :white_check_mark: |
| < v0.2  | :x:                |

## Reporting a Vulnerability

We take the security of our tools seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, follow these steps:

1. **Email us**: Send a detailed report to `hemangjoshi37a@gmail.com`.
2. **Details to include**:
   - Description of the vulnerability.
   - Steps to reproduce.
   - Potential impact.
3. **Response Time**: We will acknowledge your report within 48 hours.
4. **Resolution**: We aim to resolve all high-priority security issues within 7 days.

## Data Privacy

The Zerodha Brokerage Calculator:
- **Never** stores your trading data on any server.
- The web interface uses **LocalStorage**, which means data stays on your machine only.
- Does **not** require any API keys or personal credentials to function.

## Safe Usage

- Always ensure you are running the latest version from the official repository.
- Verify the integrity of the package if installing via PyPI.
